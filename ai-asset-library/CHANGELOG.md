# Changelog

All notable changes to the AI Asset Library will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-13

### Added
- Initial release of AI Asset Library prototype
- Home page with quick start templates
- Advanced search with faceted filtering (10+ filter categories)
- Multi-select asset curation
- AI Advisor drawer with closed-context Q&A and source citations
- Case management with export pack controls
- Sensitivity-based export restrictions (Client-safe, Internal, Restricted)
- Preview panel for asset details
- 30 mock assets covering 8 asset types
- 10 mock client cases
- Responsive design with Tailwind CSS
- Error boundary for graceful error handling
- TypeScript type safety throughout
- ESLint configuration with React best practices
- Accessibility features (keyboard navigation, ARIA labels, focus rings)

### Technical Details
- React 19 + TypeScript
- Vite build system
- React Router for client-side routing
- Context API for state management
- Tailwind CSS v3 with custom design system
- Professional code organization with constants extraction
- JSDoc comments on utility functions
- Loading states and error handling
